# 05 - 三八译码器（门级实现）

> 实验目标：使用与门和非门手工搭建一个3-8译码器，验证门级电路的正确性。


## 设计说明

本实验采用分层设计，将核心逻辑与物理适配分离：

- `decoder_3_8_gate.v`：门级核心，使用与门和非门实现译码器，输入输出均为正逻辑
- `decoder_3_8_top.v`：顶层适配层，仿真时直通，烧录时适配逻辑派 G1 的硬件特性

**本实验是最后一个使用门级电路手工搭建的模块。** 通过这个实验，我完成了对数字电路门级结构的深入理解。后续所有模块将直接使用行为级描述（`assign`、`always`、`case`），让综合器自动生成最优电路结构。


## 真值表

### 仿真模式（正逻辑）

| in[2:0] | out[7:0] |
|:---:|:---:|
| 000 | 00000001 |
| 001 | 00000010 |
| 010 | 00000100 |
| 011 | 00001000 |
| 100 | 00010000 |
| 101 | 00100000 |
| 110 | 01000000 |
| 111 | 10000000 |


## 门级实现原理

每个输出对应一种输入组合的三输入与门：
- `out[0] = ~in[2] & ~in[1] & ~in[0]`
- `out[1] = ~in[2] & ~in[1] &  in[0]`
- ...
- `out[7] =  in[2] &  in[1] &  in[0]`

每个三输入与门由两个二输入与门级联实现。


## Verilog 实现

### 核心逻辑层（`decoder_3_8_gate.v`）

```verilog
// ============================================
// 三八译码器（门级核心）
// 功能：in=000 → out[0]=1, ..., in=111 → out[7]=1
// 注：输入输出均为正逻辑
// ============================================

`include "../lib/gate_level/core_and_gate.v"
`include "../lib/gate_level/core_not_gate.v"

module decoder_3_8_gate (
    input  wire [2:0] in,
    output wire [7:0] out
);

    // 三个输入的反相信号
    wire [2:0] in_n;

    // 非门：对三个输入取反
    not_gate u_not0 (.a(in[0]), .out(in_n[0]));
    not_gate u_not1 (.a(in[1]), .out(in_n[1]));
    not_gate u_not2 (.a(in[2]), .out(in_n[2]));

    // 中间节点：每个输出由两个与门级联实现三输入与门
    wire t0, t1, t2, t3, t4, t5, t6, t7;

    // 000 -> 00000001
    and_gate u_and0_0 (.a(in_n[2]), .b(in_n[1]), .out(t0));
    and_gate u_and0_1 (.a(t0), .b(in_n[0]), .out(out[0]));

    // 001 -> 00000010
    and_gate u_and1_0 (.a(in_n[2]), .b(in_n[1]), .out(t1));
    and_gate u_and1_1 (.a(t1), .b(in[0]), .out(out[1]));

    // 010 -> 00000100
    and_gate u_and2_0 (.a(in_n[2]), .b(in[1]), .out(t2));
    and_gate u_and2_1 (.a(t2), .b(in_n[0]), .out(out[2]));

    // 011 -> 00001000
    and_gate u_and3_0 (.a(in_n[2]), .b(in[1]), .out(t3));
    and_gate u_and3_1 (.a(t3), .b(in[0]), .out(out[3]));

    // 100 -> 00010000
    and_gate u_and4_0 (.a(in[2]), .b(in_n[1]), .out(t4));
    and_gate u_and4_1 (.a(t4), .b(in_n[0]), .out(out[4]));

    // 101 -> 00100000
    and_gate u_and5_0 (.a(in[2]), .b(in_n[1]), .out(t5));
    and_gate u_and5_1 (.a(t5), .b(in[0]), .out(out[5]));

    // 110 -> 01000000
    and_gate u_and6_0 (.a(in[2]), .b(in[1]), .out(t6));
    and_gate u_and6_1 (.a(t6), .b(in_n[0]), .out(out[6]));

    // 111 -> 10000000
    and_gate u_and7_0 (.a(in[2]), .b(in[1]), .out(t7));
    and_gate u_and7_1 (.a(t7), .b(in[0]), .out(out[7]));

endmodule
```

### 顶层适配层（`decoder_3_8_top.v`）

```verilog
// ============================================
// 三八译码器 - 顶层适配层
// 功能：仿真时直通，烧录时适配硬件负逻辑
// 
// 硬件说明：
//   - in[2]、in[1] 来自按键（低电平有效），需取反
//   - in[0] 来自跳线帽（直接控制电平），不取反
//   - 数码管为共阳（低电平点亮），输出需取反
// ============================================

module decoder_3_8_top (
    input  wire [2:0] in,   // 物理输入
    output wire [7:0] out   // 物理输出
);

    wire [2:0] in_logic;
    wire [7:0] out_logic;

`ifdef SIM
    // ========================================
    // 仿真模式：正逻辑直通（波形干净）
    // ========================================
    assign in_logic = in;
    assign out      = out_logic;
`else
    // ========================================
    // 烧录模式：负逻辑适配
    //   - in[2]、in[1] 来自按键，取反（按下=0 → 逻辑1）
    //   - in[0] 来自跳线帽，不取反（直接控制）
    //   - 输出取反（数码管共阳，低电平点亮）
    // ========================================
    assign in_logic[2] = ~in[2];
    assign in_logic[1] = ~in[1];
    assign in_logic[0] = in[0];

    assign out = ~out_logic;
`endif

    // 调用门级核心译码器
    decoder_3_8_gate u_core (
        .in(in_logic),
        .out(out_logic)
    );

endmodule
```

## 设计心得

本实验的独特价值在于：

- **验证门级理解**：亲手用与门和非门搭出译码器，验证了对门级电路的理解
- **建立硬件直觉**：理解了译码器本质上是一系列与门的组合
- **学习分层设计**：将核心逻辑与物理适配分离，使核心逻辑可复用
- **完成认知跃升**：认识到后续更复杂的模块（加法器、寄存器、ALU、CPU）不可能也不应该用手搭门电路来实现，而应该用行为级描述让综合器自动生成


## 小结

- 组合逻辑电路，无时钟依赖
- 使用与门和非门实现3-8译码器
- 采用分层设计：核心逻辑层 + 物理适配层
- 本实验是最后一个门级手工搭建的模块
- 后续实验将全面转入行为级描述
- **下一实验预告**：3-8译码器（行为级实现）
