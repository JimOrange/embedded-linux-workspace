# 01 - 二输入与门（AND Gate）

> 实验目标：实现一个二输入与门。当两个输入按键同时按下时，LED 点亮。

---

## 真值表

在负逻辑下，输入为 0 表示按键按下，输出为 0 表示 LED 点亮。

| 按键 a | 按键 b | 物理电平 a | 物理电平 b | LED out |
|:---:|:---:|:---:|:---:|:---:|
| 松开 | 松开 | 1 | 1 | 灭（1） |
| 松开 | 按下 | 1 | 0 | 灭（1） |
| 按下 | 松开 | 0 | 1 | 灭（1） |
| 按下 | 按下 | 0 | 0 | 亮（0） |

> 关于负逻辑的详细说明，请参考总览 README。

---

## 逻辑表达式

物理层：`out = ~(~a & ~b)`（等价于 `out = a | b`，但在负逻辑下表现为与门功能）

> 为什么不用 `out = a & b`？在负逻辑下，`a=0, b=0` 时 `a & b = 0`，LED 亮（达到目的）。但 `a=0, b=1` 时 `a & b = 0`，LED 也亮（错误）。负逻辑下的与门需要用德摩根定律变换。

---

## Verilog 实现

```verilog
module and_gate (
    input  wire a,
    input  wire b,
    output wire out
);
    // 负逻辑下的与门：两个按键都按下（a=0, b=0）时输出 0（LED 亮）
    assign out = ~(~a & ~b);
endmodule
```

---

## 硬件验证（逻辑派 G1）

### 引脚分配

| 模块端口 | FPGA 管脚 | 连接外设 | 电平特性 |
|:---:|:---:|:---|:---|
| a | D11 | KEY0 | 低电平有效（按下为 0） |
| b | F10 | KEY1 | 低电平有效（按下为 0） |
| out | R9 | LED0 红色 | 低电平点亮（输出 0 亮） |

### 约束文件（`.cst`）

IO_LOC "a" D11;
IO_PORT "a" IO_TYPE=LVCMOS33 PULL_MODE=UP;

IO_LOC "b" F10;
IO_PORT "b" IO_TYPE=LVCMOS33 PULL_MODE=UP;

IO_LOC "out" R9;
IO_PORT "out" IO_TYPE=LVCMOS33 PULL_MODE=UP DRIVE=8;

### 验证结果

| 操作 | 预期结果 | 实际结果 |
|------|----------|----------|
| 两个按键都松开 | LED 灭 | ✅ 通过 |
| 只按 KEY0 | LED 灭 | ✅ 通过 |
| 只按 KEY1 | LED 灭 | ✅ 通过 |
| KEY0 + KEY1 | LED 亮 | ✅ 通过 |
